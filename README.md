# 📊 Student Performance Regression Analysis

## 🎯 Goal
- Identify key factors influencing overall academic performance.
- Provide actionable insights for resource allocation and policy decisions.

---

## 📂 Project Structure


---

## 📊 Key Results
- **Final Exam Score (0.35)** → strongest predictor.
- **Midterm Score (0.25)** and **Assignments (0.20)** → moderate impact.
- **Attendance (0.10)** and **Participation (0.10)** → weak influence.
- **Study Hours / Sleep Hours ≈ 0.00** → negligible impact.

---

## 🧠 Recommendations
1. Focus resources on improving final exam quality and preparation.
2. Redesign evaluation policies to better reflect participation and attendance.
3. Develop support programs for students at academic risk.
4. Leverage insights for competitive advantage in educational offerings.
