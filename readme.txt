About Dataset
This dataset contains detailed records of student academic performance along with behavioral and background factors. It is designed to support data analysis, visualization, and machine learning tasks focused on understanding the key drivers of student success.

Each record represents an individual student and includes information about study habits, attendance, assessment scores, and personal learning conditions. The dataset allows users to explore how different factors influence overall academic outcomes.

Features Overview
Academic Scores: Assignment, Midterm, Final Exam, Participation
Study Behavior: Daily study hours, sleep duration
Engagement: Attendance percentage, class participation
Background Factors: Parental education, internet access, extra classes
